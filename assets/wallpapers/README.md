# Wallpapers

For the Ambitopia wallpapers, I wanted something dark—nearly black at the top—to work with any macOS menu configuration, but also create some depth if the user has margins in their window management.

**Provided formats:**
- PNG (lossless)
- JPG (smaller file size)

**Supported resolutions:**
- [5120x2880](5120x2880/) (Desktop displays)
- [3456x2234](3456x2234/) (Laptop displays)

## Installation

### 01. Set wallpaper

Navigate to your resolution folder (3456x2234 for laptops, 5120x2880 for desktops), then right-click your variant → **Set as Desktop Picture**

**For yellow variant:**
Right-click `ambitopia-yellow.png` or `ambitopia-yellow.jpg`

**For red variant:**
Right-click `ambitopia-red.png` or `ambitopia-red.jpg`
